## Concept

A **golden cross** occurs when the fast MA crosses above the slow MA.
A **death cross** occurs when the fast MA crosses below the slow MA.

## Parameters

Accepts pre-computed fast and slow MA arrays (SMA, EMA, or any variant).

## Interpretation

| Event         | Signal                                  |
| ------------- | --------------------------------------- |
| Golden cross  | Bullish — trend turning upward          |
| Death cross   | Bearish — trend turning downward        |
| No cross      | Current trend continues                 |

<Aside type="tip">
  The classic 50/200 SMA crossover is widely watched but lags significantly.
  For faster signals, use 9/21 EMA crossovers with volume confirmation.
</Aside>

## Usage

```ts
import { detectMaCrossovers } from "../../src/domain/ma-crossover";

const events = detectMaCrossovers(fastMa, slowMa);
```
