---
title: Klinger Oscillator
description: Volume-based momentum oscillator comparing volume flow in up-trend vs down-trend periods.
---

import { Aside } from "@astrojs/starlight/components";

## Formula

$$
VF = V \times |2 \cdot \frac{dm}{cm} - 1| \times T \times 100
$$

$$
KO = EMA(VF, \text{fast}) - EMA(VF, \text{slow})
$$

where $T = +1$ if trend up, $-1$ if trend down.

## Parameters

| Parameter | Default | Range  | Notes                    |
| --------- | ------- | ------ | ------------------------ |
| `fast`    | 34      | 5–100  | Fast EMA period          |
| `slow`    | 55      | 10–200 | Slow EMA period          |

## Interpretation

| KO Value          | Signal                               |
| ----------------- | ------------------------------------ |
| KO > 0            | Net volume in uptrend                |
| KO < 0            | Net volume in downtrend              |
| Signal crossover  | Volume momentum shift                |

<Aside type="tip">
  Klinger divergences from price are strong reversal warnings —
  volume is shifting before price follows.
</Aside>

## Usage

```ts
import { computeKlingerOscillator } from "../../src/domain/klinger-oscillator";

const values = computeKlingerOscillator(candles, { fast: 34, slow: 55 });
```
