title: Parabolic SAR
description: Wilder's stop-and-reverse trailing stop that accelerates with the trend.
---

## Formula

$$
SAR_{t+1} = SAR_t + AF \times (EP - SAR_t)
$$

AF starts at `afStart` and increases by `afStep` each new extreme, capped at `afMax`.

## Parameters

| Parameter | Default | Range        | Notes                            |
| --------- | ------- | ------------ | -------------------------------- |
| `afStart` | 0.02    | 0.01–0.10    | Initial acceleration factor      |
| `afStep`  | 0.02    | 0.01–0.10    | AF increment per new extreme     |
| `afMax`   | 0.20    | 0.10–0.50    | Maximum acceleration factor      |

## Interpretation

| SAR Position       | Signal                            |
| ------------------ | --------------------------------- |
| SAR below price    | Uptrend — trail stop below price  |
| SAR above price    | Downtrend — trail stop above      |
| Flip event         | Potential trend reversal          |

<Aside type="tip">
  SAR is a pure trend system — it is always in the market (long or short).
  Pair with a trend filter to avoid whipsaws during consolidation.
</Aside>

## Usage

```ts
import { computeSarSeries } from "../../src/domain/parabolic-sar-calculator";

const sar = computeSarSeries(candles, {
  afStart: 0.02,
  afStep: 0.02,
  afMax: 0.2,
});
```
