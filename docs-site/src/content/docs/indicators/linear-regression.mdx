## Formula

$$
\hat{y} = a + bx
$$

$$
b = \frac{n\sum xy - \sum x \sum y}{n\sum x^2 - (\sum x)^2}
\qquad
a = \bar{y} - b\bar{x}
$$

$$
R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}
$$

## Parameters

Accepts arrays of values (or x, y pairs). No period parameter — fits to all supplied data.

## Interpretation

| Metric      | Meaning                                         |
| ----------- | ----------------------------------------------- |
| Slope > 0   | Prices trending upward over the window          |
| Slope < 0   | Prices trending downward                        |
| R² → 1      | Strong linear trend                             |
| R² → 0      | No consistent trend direction                   |

<Aside type="tip">
  The regression channel (±1σ and ±2σ) is useful for identifying overextended
  price moves and mean-reversion entries.
</Aside>

## Usage

```ts
import { linearRegression } from "../../src/domain/linear-regression";

const { slope, intercept, rSquared } = linearRegression(closes);
```
