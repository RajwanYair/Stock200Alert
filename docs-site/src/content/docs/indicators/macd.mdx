## Formula

$$
\text{MACD Line} = \text{EMA}_{12}(\text{close}) - \text{EMA}_{26}(\text{close})
$$

$$
\text{Signal Line} = \text{EMA}_9(\text{MACD Line})
$$

$$
\text{Histogram} = \text{MACD Line} - \text{Signal Line}
$$

## Parameters

| Parameter      | Default | Notes                 |
| -------------- | ------- | --------------------- |
| `fastPeriod`   | 12      | Short EMA window      |
| `slowPeriod`   | 26      | Long EMA window       |
| `signalPeriod` | 9       | Signal line smoothing |

## Interpretation

| Condition                 | Signal                       |
| ------------------------- | ---------------------------- |
| MACD crosses above Signal | Bullish crossover → **BUY**  |
| MACD crosses below Signal | Bearish crossover → **SELL** |
| Histogram flips positive  | Momentum building upward     |
| Divergence with price     | Potential reversal warning   |

<Aside type="tip">
  CrossTide's MACD method fires a **BUY** on a bullish MACD/Signal crossover when both lines are
  below zero (classic trend reversal), and **SELL** on bearish crossover when both are above zero.
</Aside>

## Usage

```ts
import { computeMacd } from "../../src/domain/macd-calculator";
import type { DailyCandle } from "../../src/types/domain";

const candles: DailyCandle[] = [
  /* OHLCV data */
];
const result = computeMacd(candles, {
  fastPeriod: 12,
  slowPeriod: 26,
  signalPeriod: 9,
});

console.log("MACD:", result.macd.at(-1));
console.log("Signal:", result.signal.at(-1));
console.log("Histogram:", result.histogram.at(-1));
```
