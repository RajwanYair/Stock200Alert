---
title: Choppiness Index
description: Measures whether the market is trending or range-bound, scaled 0–100.
---

import { Aside } from "@astrojs/starlight/components";

## Formula

$$
CI = \frac{\log_{10}\!\left(\frac{\sum ATR(1, n)}{H_n - L_n}\right)}{\log_{10}(n)} \times 100
$$

## Parameters

| Parameter | Default | Range  | Notes                    |
| --------- | ------- | ------ | ------------------------ |
| `period`  | 14      | 2–100  | Lookback window          |

## Interpretation

| CI Value | Market Condition                            |
| -------- | ------------------------------------------- |
| > 61.8   | Choppy / range-bound                        |
| 38.2–61.8| Transitioning                               |
| < 38.2   | Strongly trending                           |

<Aside type="tip">
  Use CI as a regime filter: enable trend-following strategies when CI < 38,
  and mean-reversion strategies when CI > 62.
</Aside>

## Usage

```ts
import { computeChoppinessIndex } from "../../src/domain/choppiness-index";

const values = computeChoppinessIndex(candles, { period: 14 });
```
