---
title: KAMA — Kaufman Adaptive Moving Average
description: Self-adjusting MA that speeds up in trends and slows down in choppy markets.
---

## Formula

$$
ER = \frac{|\text{Close}_t - \text{Close}_{t-n}|}{\sum_{i=1}^{n} |\text{Close}_i - \text{Close}_{i-1}|}
$$

$$
SC = \left[ER \cdot \left(\frac{2}{f+1} - \frac{2}{s+1}\right) + \frac{2}{s+1}\right]^2
$$

$$
KAMA_t = KAMA_{t-1} + SC \cdot (\text{Close}_t - KAMA_{t-1})
$$

## Parameters

| Parameter | Default | Range  | Notes                    |
| --------- | ------- | ------ | ------------------------ |
| `period`  | 10      | 2–100  | Efficiency ratio window  |
| `fast`    | 2       | 2–10   | Fast constant periods    |
| `slow`    | 30      | 10–50  | Slow constant periods    |

## Interpretation

| KAMA Behaviour       | Signal                               |
| -------------------- | ------------------------------------ |
| KAMA flat (low ER)   | Choppy market — avoid trading        |
| KAMA moving (high ER)| Trending — follow direction          |

<Aside type="tip">
  KAMA automatically avoids whipsaws in sideways markets without
  lagging in strong trends — an ideal replacement for plain EMA.
</Aside>

## Usage

```ts
import { computeKama } from "../../src/domain/kama";

const kama = computeKama(closes, { period: 10, fast: 2, slow: 30 });
```
