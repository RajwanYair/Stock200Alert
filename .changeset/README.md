# Changesets

Hello and welcome! This folder has been created by [@changesets/cli](https://github.com/changesets/changesets/tree/main/packages/cli), a build tool that works with multi-package repos, or single-package repos to help you version and publish your code.

You can find the full documentation for it in our repository.

We have a quick list of common questions to get you started working with changesets:

- [I am new to changesets, how do I get started?](https://github.com/changesets/changesets/blob/main/docs/adding-a-changeset.md)
- [I need to do a major version bump, how do I do that?](https://github.com/changesets/changesets/blob/main/docs/modifying-changelog-format.md)
- [I need to change the format of my changelog?](https://github.com/changesets/changesets/blob/main/docs/modifying-changelog-format.md)
